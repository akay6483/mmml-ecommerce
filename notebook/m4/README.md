E-Commerce & Retail Analytics Using Machine Learning

This project demonstrates how Machine Learning regression techniques can be applied in the E-commerce domain to analyze customer behavior and predict spending patterns. The goal is to support better business decisions such as personalized recommendations, targeted marketing, and customer value prediction.

The study includes implementation and comparison of three regression-based models:
- Linear Regression
- Ridge Regression (L2 Regularized Linear Model)
- Decision Tree Regressor
